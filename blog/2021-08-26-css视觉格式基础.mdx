## 视觉格式基础

不管什么元素，css都假定每个元素生成一个或者多个矩形框，我们称之为`元素框`

各个元素框的中心是`内容区域`。`内容区`四周有可选的`内边距padding`、`外边距margin`、边框、轮廓。![图像 2](https://tva1.sinaimg.cn/large/008i3skNgy1gqq38wk67gj310h0dpn0r.jpg)

### 重要概念

#### 常规流动

即从左到右，从上到下。多数元素都采用常规流动，除非原始浮动了，定位了，放到弹盒或者采用了栅格布局。

#### 非置换元素

内容包含在文档中的元素，比如段落p是非置换元素，因为段落中的文本内容在元素自身中。

#### 置换元素

为其他内容占位的元素，比如img，他指向一个图像文件，那个图像文件插入到文档流中的位置上。

#### 根元素

位于文档树顶端的元素。

#### 块级框

段落、标题或者div等元素生成的框。display:block能力把任何元素变成块级框。

块级框前后都是换行的。

#### 行内框

Strong、span等生成框体。行内框前后不换行。display:inline能把任何元素变为行内框。

#### 行内块级框

内部像块级框，外部像行内框。



### 容纳块

每个元素的框体都相对容纳块放置，简单来说，容纳块就是元素框体的“布局上下文”。

容纳块是由离元素最近的那个生成列表项或者块级框

### 调整元素的显示方式

影响浏览器显示元素的方式。

```css
display: inline | block  | inline-block .....
```

### 块级框

![image-20210521163851515](https://tva1.sinaimg.cn/large/008i3skNgy1gqq4zzy7zbj30tw0lub29.jpg)

#### box-sizing

用于改变width,height的含义

```css
box-sizing: content-box | padding-box| border-box
content-box: 默认值，width和height为内容区的大小
padding-box:width和height为内容区+padding的大小
border-box:width和height为内容区 + padding + border的大小
```

### 横向格式化属性

有7个属性影响着块级框的横向布局

margin-left、border-left、padding-left、width、padding-right、border-right、margin-right。

这7个属性的值加起来要等于元素容纳块的宽度。而这一宽度通常为块级元素的父元素的width。

这7个值中只有3个能设置为auto。margin-left， margin-right，width。其他要么为具体的值，要么为默认值0。

如果3个都是具体的值，如果不满足约束，那么margin-right会被强制设为auto。

有1个为auto，那么那个值自动计算，

有2个值为auto：

1.width固定，两个margin为auto：两个auto相等，width居中，计算两个margin

2.一个margin，一个width：此时auto的margin会被设置为0，width还是auto

有3个都是auto: 两侧auto的margin会被设置为0，width为auto

横向的外边距不折叠，所以父元素的内边距，边框，外边距都可能影响子代。

### 负外边距

内边距，width，border不能为负数，但是margin可能为负数。

但不管怎么样最终margin +border+padding +width总是等于父元素的宽度。

我们来看看下面的

```css
<html>
    <style>
        div {
            background-color: red;
            width: 500px;
            height: 100px;                    
        }
        p {
            background-color: green;
            margin-left: 0;
            width: auto;
            margin-right: -200px;
            height: 50%;
        }
    </style>
    <body>
        <div >
            <p >A paragaph</p>
        </div>
    </body>
</html>
```

![image-20210521171510618](https://tva1.sinaimg.cn/large/008i3skNgy1gqq61p1rz1j30ua0akaa7.jpg)

子元素竟然比父元素还宽了，但这个是正确符合规范的。

### 百分数

百分数是相对于容纳块的宽度。

边框不能设置为百分数。

### 置换元素

上面讲的都是常规流动模式下的非置换块级元素的横向格式化。块级置换元素也类似，有个例外是，如果width为auto，那么置换元素的width等于内容自身的宽度。

### 纵向格式化属性

和横向一样margin + padding + border + height要等于容纳块的高度，通常这是块级框父元素的height。

这7个值中也只有margin-top, margin-bottom ,height能够为auto，其他都只能大于等于0.

和横向不一样的是，如果设置margin-top或者margin-bottom为auto，那么他们将自动设置为0。（定位元素和flex元素对设置为auto的上下外边距处理方式有所不同）。

### 百分数高度

百分数是相对于容纳块的。

边框不能设置为百分数。
