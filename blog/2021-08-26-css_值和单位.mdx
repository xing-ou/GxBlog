## 关键字、字符串和其他文本值

### 关键字

none

inherit ：把属性设置为和父元素一样的值

initial：将属性值设置为预定义的初始值

unset :是inherit和intial的通用替身，如果能够继承，那么就是inherit，否则就是initial。

### 字符串

单引号，或者双引号引起来的字符序列。

```css
"sss"
'ddd'
```

### url

```css
background-image: url("xxxx.png")
```

### 图像

下面的可以代表图像

```css
<url>
<image-set>
<gradient>
```

## 距离

### 绝对长度

```css
英寸(in)
厘米(cm)
毫米(mm)
四分之一毫米(q)
点(pt) ：印刷度量单位， 1英寸有72点
派卡(pc): 也是印刷术语， 1pc等于12点
像素(px)
```

### 分辨率单位

```css
点每英寸 dpi
点每厘米 dpcm
点每像素 dppx
```

### 相对长度单位

```css
em :  1em等于font-size。理论上等于小写字母m的宽度，em的名字也由此而来
ex : 指小写字母x的高度
rem: 根元素的font-size
ch : 一个字符，等于渲染时所用字体中“0“字形的进距。
vm : 把视区宽度100等分。每一份就是1vm
vh ： 把视区高度100等分，每一份就是1vh
vmin : min(vm, vh)
vmax : max(vm, vh)
```

## 计算值

为了方便计算，css提供了calc()， 你可以进行些数学计算

```css
p {
  width : calc(100% - 10px)
}
```

## 属性值

使用attr()， 可以引用某个属性的值

```css
div::before {
            content: "[" attr(class) "]";
        }
```

## 自定义值

```css
html {
  --my-color: red;

}

p {
  color: var(--my-color);
}
自定义值必须以--开头，通过var引用。

```

变量是有作用域的

```css
html {
    --my-color: green;
}  

div {
    --my-color: yellow;
}

div {
  color: var(--my-color);//yellow
}

```
